# Vue 3

## Installation

```bash
pnpm add @hemia/lume-vue
pnpm add -D hemia-lume
```

## Initialize

```bash
bunx --bun hemia-lume@latest init
# Detects Vue automatically from package.json
```

## Add a component

```bash
bunx --bun hemia-lume@latest add button
```

Result:
```
src/components/ui/button/
├── button.vue
└── button.variants.ts
```
